总结

需支持跨域请求

根据演示示例，可能需要如下数据表

表1：

用于保存已设计好的表单，接口3提交保存到此表；为接口4提供已保存的数据

表1必需字段

字段 说明
id 当前记录id，可为自动生成
formId 表单数据源id，即使用了哪个数据源创建的表单，当表单新增数据时根据此id提交到对应的表
name 创建的表单名称
formData 设计的表单json字符串数据
searchData 表单列表搜索表单字符串数据
tableData 表单列表字符串数据

表2

用作本次演示的数据表，每张表对应一个表单数据。为接口2提供当前所有可供数据录入的字段；为接口1提供可用于设计的表单数据源

表2演示字段

字段 说明
文本框 表单提交的字段
多选框 同上
单选 同上
多选 同上
… 可提供所有可设计的字段用于测试
