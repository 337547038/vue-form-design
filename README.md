# Vue自定义表单设计器

## 项目结构
```txt
public
  ├─assets 静态资源文件
  └─index.html // 模板页面
src
  ├─assets // api文件
    └─scss // 样式文件
  ├─packages // 组件资源包
  ├─router // 路由配置
  ├─store // 状态管理
  ├─utils // 工具类
  └─views // 页面
      ├─design // 表单设计页面
      ├─form // 表单数据添加,修改,预览页
      ├─list // 数据列表页
      └─index // 页面入口，表单列表页
```
